# Gallery Images

Place all site images in this directory. 

## Naming Convention

```
hero-[descriptor].jpg          # Hero/banner images
collection-[category].jpg      # Category cover images  
artwork-[id]-[title].jpg       # Individual artwork photos
rachel-[descriptor].jpg        # Artist portrait/studio shots
mural-[id]-[name].jpg          # Mural location photos
```

## Required Images (populate as available)

- `hero-main.jpg` — Homepage hero background
- `rachel-portrait.jpg` — Artist portrait for Story page
- `rachel-studio.jpg` — Studio shot  
- `collection-beach-coastal.jpg` — Beach & Coastal category cover
- `collection-sea-life.jpg` — Sea Life category cover
- `collection-birds-wildlife.jpg` — Birds & Wildlife category cover
- `collection-florals.jpg` — Florals category cover
- `collection-abstracts.jpg` — Abstracts category cover
- `collection-palm-trees.jpg` — Palm Trees category cover
- `collection-mermaids-whimsy.jpg` — Mermaids & Whimsy category cover
- `collection-watercolors.jpg` — Watercolors category cover
- `collection-line-art.jpg` — Line Art category cover
- `mural-[1-14]-[name].jpg` — Each of the 14 mural locations

## Image Specs

- Hero images: 1920×1080px minimum, JPEG (quality 85)
- Collection covers: 800×1000px (4:5 ratio), JPEG (quality 85)
- Artwork photos: 1200×1500px maximum, JPEG (quality 90)
- Artist photos: 800×1000px minimum, JPEG (quality 85)
- Mural photos: 1200×900px minimum, JPEG (quality 85)

All images should be optimized before placing here.
Use `next/image` with width/height props to enable Next.js automatic optimization.
